//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Объект задает структуру данных и внутреннюю реализацию работы с ними.
Источники данных:
1. Реляционная база данных.
2. Форма.
3. Бинарные данные.
4. XML.

У объекта есть 2 функции, котрые служат для импорта и экспорта данных:
1. import - импорт данных из источника в объект (по сути старый добрый Create)
	например: udResult Import( const udDataSource& source )
	
2. export - экспорт данных из объекта в формат источника данных
	например: udResult Export( udDataSource& source )

Для импорта и экспорта данных, каждый формат должен обладать неким интерфейсом, чтобы транспортировать данные.
Интерфейс должен обладать возможностью импорта и экспорта значения атрибута объекта:
1. udResult SetAttribute( const string& szAttributeName, const udPType pValue ) - экспорт атрибута объекта в источник.
2. udResult GetAttribute( cosnt string& szAttributeName ) const - импорт атрибута объекта из источника.
